# Zork 1 reference bootstrap

This directory contains a DjangoMOO bootstrap of *Zork I: The Great Underground
Empire*. It is loaded via:

```bash
manage.py moo_init --bootstrap zork1
```

The bootstrap creates 110 rooms, 122 objects, an inheritance hierarchy of
Zork-flavored classes (`$zork_room`, `$zork_thing`, `$zork_container`, etc.),
and the supporting `$zil_sdk` machinery (turn counter, randomness, message
queue, state tables) on the system object.

## Provenance

This is a **derivative work**, not original content. The Python files in this
directory (rooms, objects, exits, verbs) were generated by the
[`extras/zil_import/`](../../../extras/zil_import/) converter from Infocom's
`dungeon.zil` and `actions.zil` — the original 1980 Zork 1 source code.

In 2025, Microsoft / Activision Publishing, Inc. released that source code
under the MIT License at
<https://github.com/the-infocom-files/zork1>. This bootstrap inherits and
preserves those terms; see [`LICENSE`](LICENSE) in this directory for the
verbatim upstream notice.

## Why it ships with django-moo

The `zork1` dataset is a reference/educational example. It exists to
demonstrate DjangoMOO's verb dispatch, parent-child object inheritance,
turn-based clock and probability tables, and the `--bootstrap <name>` model
for swapping in alternate worlds. It is not the project's main purpose; the
default world is loaded with `manage.py moo_init` (no `--bootstrap`).
